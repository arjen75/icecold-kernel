    Hello and welcome

    Some informations about this kernel project

    This kernel is based on googles 3.0.31 android kernel. All the needed modifactions for the P500 (e.g. drivers) are ported from LGE 2.6.35 kernel.

    What works

    Nothing! For the moment there are lot of compilation errors needed to be fixed and drivers to port.
